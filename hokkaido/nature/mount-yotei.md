# 羊蹄山 (Mount Yōtei)

## 基本資訊

| 項目 | 內容 |
|------|------|
| 日文名稱 | 羊蹄山 |
| 海拔 | 1,898 公尺 |
| 特色 | 完美圓錐山形、蝦夷富士、滑雪勝地 |

## 景點介紹

羊蹄山是北海道最具標誌性的山峰之一，因其優美的圓錐形山體與富士山相似，被當地人暱稱為「蝦夷富士」（Ezo Fuji）。這座活火山位於新雪谷和俱知安町附近，海拔1,898公尺，雖然不如旭岳高大，但因其完美的山形而成為北海道最風景如畫的山峰之一，被選為日本100名山之一。

羊蹄山一年四季都有不同的魅力：冬季時，山體被積雪覆蓋，成為著名的滑雪聖地，新雪谷和留壽都等世界級滑雪度假村就坐落在山腳；夏季時，山地是登山和戶外活動的天堂，徒步登山的路線風景優美；秋季時，周圍的樹林變成金紅色，與山景構成一幅經典的北海道風景畫。

從山頂可以眺望附近的尻別湖和和日本海，天氣晴朗時還能看到更遠的太平洋。最佳的登山季節是7月到10月，其中8月的杜鵑花季和9-10月的紅葉季是最佳時節。山下的小鎮如俱知安和新雪谷有許多溫泉旅館和餐廳，是體驗北海道鄉村風情的好去處。

## 詳細地址
